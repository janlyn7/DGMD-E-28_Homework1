Where I am From

I am originally from Jamaica, but grew up Houston, TX. I came to Boston for university and ended up staying.

My Major

I'm not enrolled in any University program, so I don't have a major. I'm taking this class to increase my knowledge of modern web technologies and frameworks and to strengthen my web programming skills.

Programming Experience

I've been a programmer for over 20 years, starting with IDL, moving onto C/C++, Perl, Java, and Python, with a smattering of HTML, CSS, and Javascript when needed for internal webpages.

I've previously taken Fundamentals of Website Development and Dynamic Web Applications at the Harvard Extension School, both of which I thoroughly enjoyed.

Over the past year, I've assumed responsibility for a suite of web-based applications, developing the front-end and middleware for two new web tools for our users. However, these are all legacy applications operating within an ancient framework that still uses Java, JSP, and Perl. I'm hoping this class help me learn how to modernize these web applications for the 21st century.
